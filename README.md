# Gerador de Senhas

Um gerador de senhas desenvolvida em Python utilizando a biblioteca Tkinter, proporcionando senhas seguras.

## Funcionalidades
- Selecionar mínimo e máximo de caracteres
- Senhas aleatórias com letras, números e caracteres especiais.

## Tecnologias utilizadas
- Python 3.11.9
- Tkinter

## Como executar o projeto
- Acesse o projeto pelo código fonte e faça dowload
- Abra o VSCode na pasta do projeto
- Execute GeradorDeSenhas.py
